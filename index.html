My Current Projects

Spotify2YT

Spotify2YT is a Web App that converts your Spotify playlist to a YouTube playlist! It was made in Typescript and is waiting to be approved for public use!

Moore

Moore is a game I created using the Unity game engine and C#. It has multiple playable characters drawn by people in my programming class. You just jump on platforms and hope you do not fall. It is a basic Unity game to test my game development skills.

Puck

Puck is a multi-purpose Discord bot that is the first major programming project I have taken part in. It could play music in voice chats and enhance the fun in Discord servers with commands such as love and 8ball. I still host this bot after 3 years of first creating it. It is still actively used in a few Discord servers.

noblox.js

Noblox.js is an open source Roblox API wrapper for Node.js. It is a huge project utilized by thousands of developers and has over 1,000 weekly downloads. I contributed by adding code and typings for a new API endpoint that was added to Roblox.
